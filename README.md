# RedeCalc Pro Profissional

## Recursos adicionados
- Prancha planimétrica melhorada
- Escala gráfica automática
- Coordenadas UTM aproximadas
- Memorial descritivo em PDF
- Exportação DXF básica para AutoCAD/QGIS
- Tabela de pontos com latitude, longitude, UTM, poste, estruturas e vãos
- Lista consolidada de materiais
- Rede MT tracejada e rede BT contínua
- Distância total e distância por trecho

## Uso
1. Publique no GitHub Pages.
2. Capture os pontos com GPS.
3. Informe os dados da prancha.
4. Gere a prancha PDF, memorial PDF ou DXF.

## Observação
As coordenadas UTM são aproximadas via cálculo local no navegador. Para projeto executivo, validar em software topográfico/GIS.
